Sistema de Gestión de Tareas y Animales con Node.js, Express y Vanilla JS

Este proyecto es un sistema web simple pero funcional, desarrollado para gestionar el registro de animales y la creación de tareas. El backend está construido con Node.js y Express.js, proporcionando una API RESTful que maneja la lógica del negocio. La persistencia de los datos se gestiona a través de una base de datos SQL administrada con phpMyAdmin, donde se almacenan la información de los animales y las tareas.

API CRUD con Node.js, Express y Mongoose para MongoDB

Este es un proyecto simple que creé para practicar mis habilidades. Es una API hecha con Node.js y Express que me permite hacer las operaciones básicas de CRUD (Crear, Leer, Actualizar y Eliminar). Para guardar los datos, usé una base de datos MongoDB y la conecté con la librería Mongoose, lo que me ayudó a organizar la información fácilmente.

API RESTful de Tareas con Node.js, Express y Sequelize

En este proyecto hice una API para manejar tareas. Usé Node.js y Express para crear el servidor. Lo más importante que aprendí fue a usar Sequelize, que me ayudó a conectar la API con una base de datos SQL sin tener que escribir las consultas a mano.

El proyecto me sirvió para practicar cómo se hacen las operaciones CRUD (Crear, Leer, Actualizar, Eliminar) y entender cómo funciona una API con una base de datos relacional de forma organizada.
